# Critérios de Entrada e Saída

## Entrada (para iniciar testes)

- [ ] Código mergeado na branch de desenvolvimento
- [ ] Build passando no CI
- [ ] Deploy realizado no ambiente de QA
- [ ] Critérios de aceite documentados na story
- [ ] Dados de teste disponíveis

## Saída (para considerar pronto)

- [ ] Todos os critérios de aceite verificados
- [ ] Nenhum bug crítico ou alto em aberto
- [ ] Testes de regressão passando
- [ ] Casos de teste documentados ou automatizados

## Saída de Sprint

- [ ] Suite de regressão com pass rate ≥ 95%
- [ ] Zero bugs críticos em aberto
- [ ] Relatório de teste gerado e compartilhado
- [ ] Smoke test em homologação aprovado
